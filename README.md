Sistema de Gestión Departamental Multi-Almacén
CBTIS 49 - Proyecto de Desarrollo Web

Integrante: Edsson Leal

Descripción
Aplicación web funcional para el control de inventarios distribuidos en 5 almacenes, con módulos de ventas, compras y generación de reportes inteligentes.

Tecnologías utilizadas:
HTML5 / CSS3: Maquetación y diseño responsivo.
JavaScript: Lógica de inventario y procesamiento de transacciones.
Git/GitHub: Control de versiones y despliegue.
